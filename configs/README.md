# Configuration Files

This directory contains sample configuration files used in the Splunk-Wazuh Home Lab.

## Files

| File | Description |
|------|-------------|
| `inputs.conf` | Configures log collection for the Splunk Universal Forwarder. |
| `outputs.conf` | Defines the destination Splunk server for forwarding logs. |
| `ossec.conf` | Main configuration file for the Wazuh Manager. |
| `agent.conf` | Centralized configuration for Wazuh agents. |

## Note

Sensitive information such as IP addresses, passwords, authentication keys, and tokens has been removed or replaced with placeholders.

These files are intended for educational purposes and demonstrate the configuration used in this home lab.
